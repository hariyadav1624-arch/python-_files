# Python Mini Projects

A collection of beginner-to-intermediate Python mini projects created to improve programming skills, problem-solving abilities, and understanding of Python concepts through hands-on practice.

## Overview

This repository contains various Python projects covering topics such as:

- Python Fundamentals
- Object-Oriented Programming (OOP)
- File Handling
- Data Structures
- Algorithms
- Automation
- Games
- Utilities
- Command-Line Applications

Each project is designed to reinforce programming concepts while building practical applications.

## Requirements

- Python 3.8 or higher

Check your Python version:

```bash
python --version
```

## How to Run a Project

Navigate to the project directory:

```bash
cd Calculator
```

Run the Python file:

```bash
python main.py
```

## Featured Projects

### Calculator
A simple calculator that performs basic arithmetic operations.

### Number Guessing Game
A game where the user guesses a randomly generated number.

### Password Generator
Generates secure random passwords based on user preferences.

### To-Do List
A command-line task manager for adding, viewing, and removing tasks.

### Expense Tracker
Tracks daily expenses and generates summaries.

### Weather App
Fetches real-time weather information using an API.
